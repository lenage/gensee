# Gensee

Gensee is Ruby wrapper for [Gensee API](http://www.gensee.com/document-technical.php),

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'gensee'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gensee
